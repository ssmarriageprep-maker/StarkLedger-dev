# Android Tools Usage

ALWAYS use the ADB binary located at `~/.gemini/antigravity/tools/platform-tools/adb` for any mobile deployment or debugging tasks.
Do NOT use the system `adb` or attempt to install it via brew or other package managers.
The path to `adb` is fixed and known.
